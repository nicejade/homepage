## 项目介绍

Continue 是面向开发者的开源 AI 编程助手，既可以作为 VS Code、JetBrains 等环境中的日常插件，也可以承载团队自己的模型、提示词和上下文策略。它适合希望把 AI 编程能力留在可控工具链里的团队。

## GitHub 仓库预览

仓库位于 [continuedev/continue](https://github.com/continuedev/continue)，核心语言为 TypeScript，内容覆盖编辑器扩展、命令行能力和代理配置。项目主页提供安装入口、文档和团队使用说明。

## 对用户价值

Continue 的价值在于可组合：开发者可以选择模型、配置上下文来源，并把重复性的编码、解释和重构任务放进熟悉的 IDE 流程里。对于关注隐私和可控性的团队，它比完全黑盒的托管工具更容易评估和治理。

## 适应人群

- 希望在 IDE 内使用 AI 编程助手的个人开发者。
- 需要统一团队编码代理配置的工程团队。
- 想把本地模型或私有模型接入开发流程的 AI 工具探索者。

## 如何使用

从项目主页或编辑器市场安装 Continue，选择模型提供方，然后在项目中补充规则、上下文和命令。团队使用时建议先用一个小仓库验证模型、上下文和权限边界，再逐步推广到主力代码库。
