## 项目介绍

n8n 是一个工作流自动化平台，支持可视化编排、代码节点、自托管和大量集成。随着原生 AI 能力加入，它也成为把模型能力接入日常业务流程的常用选择。

## GitHub 仓库预览

仓库位于 [n8n-io/n8n](https://github.com/n8n-io/n8n)，核心语言为 TypeScript。项目主题覆盖 automation、integrations、MCP、self-hosted、workflow automation 和 low-code。

## 对用户价值

n8n 的优势在于连接器和流程表达力：用户可以把表单、数据库、Webhook、模型调用和通知系统连成自动化链路。相比只写脚本，它更适合团队协作、调试和维护。

## 适应人群

- 需要把多个 SaaS 或内部系统串起来的运营和工程团队。
- 想自托管自动化平台的用户。
- 希望把 AI 节点接入业务流程的自动化实践者。

## 如何使用

从官方文档选择云服务或自托管部署，先用一个低风险流程验证凭据、触发器和错误处理。涉及生产数据时，应单独管理密钥、权限和执行日志。
