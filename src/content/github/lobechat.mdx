## 项目介绍

LobeChat 是一个开源的现代化 AI 聊天框架，支持 OpenAI、Anthropic、Google 等云端模型供应商，也可以对接 Ollama 等本地模型。它内置插件市场、知识库上传和 TTS/STT 能力，适合作为个人或团队的 AI 交互入口。

## GitHub 仓库预览

仓库位于 [lobehub/lobe-chat](https://github.com/lobehub/lobe-chat)，核心语言为 TypeScript。项目提供 Docker 一键部署、PWA 支持和丰富的模型配置面板，整体交互设计在开源项目中属于第一梯队。

## 对用户价值

LobeChat 为多模型切换提供了统一界面，避免在不同服务之间反复登录和复制粘贴。插件和知识库功能适合需要长期积累对话上下文或接入外部工具的用户。

## 适应人群

- 需要统一管理多模型访问的个人用户。
- 希望自建团队 AI 聊天前端的工程团队。
- 关注聊天 UI 交互质量的产品和设计参考者。

## 如何使用

通过 Docker 或 Vercel 一键部署，配置模型服务商的 API 密钥即可使用。可选的本地模型对接通过 Ollama 端点实现。多用户场景建议配合外部数据库部署。
