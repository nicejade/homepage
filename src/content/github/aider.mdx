## 项目介绍

Aider 是一个在终端中使用的 AI 结对编程工具，可以读取代码库上下文、生成补丁并配合 Git 工作流迭代。它适合习惯命令行、希望保留清晰 diff 审查节奏的开发者。

## GitHub 仓库预览

仓库位于 [Aider-AI/aider](https://github.com/Aider-AI/aider)，核心语言为 Python。项目主页提供安装、模型配置和使用示例，主题集中在 CLI、OpenAI、Claude、Gemini 和代码协作。

## 对用户价值

Aider 的价值在于低摩擦：开发者可以在当前仓库直接提出修改目标，查看模型生成的变更，再决定是否接受。它把对话、文件编辑和 Git 反馈环放在一个终端流程里。

## 适应人群

- 偏好终端和 Git 工作流的开发者。
- 想用 AI 辅助局部重构、测试和文档的人。
- 需要快速比较不同模型编码能力的技术用户。

## 如何使用

安装 Aider 后进入 Git 仓库，配置模型密钥并指定要编辑的文件。建议从小范围任务开始，例如补测试、修文档或重构单个函数，并在每轮后检查 diff。
