йЎ№зӣ®д»Ӣз»Қ

Puppeteer жҳҜ Google Chrome еӣўйҳҹз»ҙжҠӨзҡ„ Node.js еә“пјҢйҖҡиҝҮ DevTools Protocol жҲ– WebDriver BiDi д»Ҙй«ҳеұӮ API й©ұеҠЁ Chrome жҲ– FirefoxгҖӮй»ҳи®Өд»Ҙж— еӨҙпјҲheadlessпјүжЁЎејҸиҝҗиЎҢпјҢд№ҹж”ҜжҢҒжңүз•Ңйқўи°ғиҜ•гҖӮд»“еә“дёә monorepoпјҢж—ҘеёёејҖеҸ‘е®үиЈ… npm еҢ… puppeteer жҲ– puppeteer-core еҚіеҸҜпјӣе®ҳж–№ж–ҮжЎЈдёҺ API еҸӮиҖғи§Ғ pptr.devгҖӮ

ж ёеҝғзү№жҖ§

  • й«ҳеұӮжөҸи§ҲеҷЁжҺ§еҲ¶ APIпјҡlaunchгҖҒnewPageгҖҒgotoгҖҒзӮ№еҮ»гҖҒй”®зӣҳиҫ“е…ҘгҖҒжҲӘеӣҫгҖҒPDF еҜјеҮәзӯүеёёи§ҒиҮӘеҠЁеҢ–иғҪеҠӣејҖз®ұеҚіз”Ё
  • Chrome дёҺ Firefox еҸҢеј•ж“ҺпјҡChrome йҖҡиҝҮ DevTools ProtocolпјӣFirefox иө° WebDriver BiDiпјҢи·ЁжөҸи§ҲеҷЁеңәжҷҜеҸҜз»ҹдёҖи„ҡжң¬з»“жһ„
  • puppeteer / puppeteer-core еҸҢеҢ…зӯ–з•Ҙпјҡе®Ңж•ҙеҢ…е®үиЈ…ж—¶дёӢиҪҪеҢ№й…ҚзүҲжң¬жөҸи§ҲеҷЁпјӣcore еҢ…д»…дҪңеә“дҪҝз”ЁпјҢйңҖиҮӘиЎҢжҢҮе®ҡ executablePath жҲ–й…ҚеҗҲ @puppeteer/browsers з®ЎзҗҶдәҢиҝӣеҲ¶
  • Locator дёҺеҸҜи®ҝй—®жҖ§йҖүжӢ©еҷЁпјҡж”ҜжҢҒ ::-p-aria()гҖҒ::-p-text() зӯүиҜӯд№үеҢ–е®ҡдҪҚпјҢеҮҸе°‘ brittle CSS йҖүжӢ©еҷЁдҫқиө–
  • жөҸи§ҲеҷЁдёӢиҪҪдёҺй…ҚзҪ®дҪ“зі»пјҡnpx puppeteer browsers installгҖҒ.puppeteerrc.js дёҺ PUPPETEER_CACHE_DIR зӯүжңәеҲ¶дҫҝдәҺ CI дёҺе®№еҷЁзҺҜеўғеӨҚзҺ°
  • з”ҹжҖҒе»¶дјёпјҡе®ҳж–№жҺЁиҚҗ chrome-devtools-mcp дҪңдёәеҹәдәҺ Puppeteer зҡ„ MCP жңҚеҠЎеҷЁпјӣе№¶е®һйӘҢжҖ§ж”ҜжҢҒ WebMCP

еҜ№з”ЁжҲ·д»·еҖј

иӢҘдҪ йңҖиҰҒеңЁ Node.js дёӯзЁӢеәҸеҢ–жү“ејҖзҪ‘йЎөгҖҒеЎ«иЎЁгҖҒжҲӘеӣҫгҖҒз”ҹжҲҗ PDFгҖҒжҠ“еҸ–еҠЁжҖҒжёІжҹ“еҶ…е®№жҲ–и·‘з«ҜеҲ°з«ҜеӣһеҪ’пјҢPuppeteer жҸҗдҫӣдәҶжҜ”зӣҙжҺҘи°ғ DevTools жӣҙдҪҺй—Ёж§ӣзҡ„жҠҪиұЎгҖӮе®үиЈ… puppeteer ж—¶дјҡжӢүеҸ–е…је®№зүҲ ChromeпјҢжң¬ең°дёҺ CI зҺҜеўғдёҖиҮҙжҖ§иҫғеҘҪпјӣеңЁ Docker жҲ–еҸ—йҷҗзҪ‘з»ңдёӢпјҢеҸҜж”№з”Ё puppeteer-core жҢӮиҪҪе·Іжңү ChromiumпјҢжҲ–йҖҡиҝҮй…ҚзҪ®зј“еӯҳзӣ®еҪ•йҒҝе…ҚйҮҚеӨҚдёӢиҪҪгҖӮеҜ№ AI Agent дёҺиҮӘеҠЁеҢ–е№іеҸ°пјҢPuppeteer д№ҹжҳҜ chrome-devtools-mcp зӯүгҖҢжөҸи§ҲеҷЁе·Ҙе…·гҖҚзҡ„еёёи§Ғеә•еұӮе®һзҺ°пјҢйҖӮеҗҲжҠҠгҖҢи®©жЁЎеһӢж“ҚдҪңзңҹе®һжөҸи§ҲеҷЁгҖҚжҺҘе…Ҙе·ҘдҪңжөҒгҖӮ

дёҺжӣҝд»Јж–№жЎҲ

  • зӣёжҜ” PlaywrightпјҢPuppeteer дёҺ Chrome з”ҹжҖҒз»‘е®ҡжӣҙж·ұгҖҒеҺҶеҸІжӣҙд№…пјҢStars дёҺ npm дёӢиҪҪйҮҸеәһеӨ§пјӣPlaywright еҶ…зҪ®еӨҡжөҸи§ҲеҷЁе№¶иЎҢгҖҒиҮӘеҠЁзӯүеҫ…дёҺжӣҙејәзҡ„дёҖдҪ“ејҸжөӢиҜ• RunnerпјҢж–°йЎ№зӣ®еҒҡи·ЁжөҸи§ҲеҷЁ E2E ж—¶ Playwright еҫҖеҫҖжӣҙзңҒеҝғгҖӮ
  • зӣёжҜ” SeleniumпјҢSelenium еҹәдәҺ WebDriver ж ҮеҮҶгҖҒеӨҡиҜӯиЁҖе®ўжҲ·з«ҜжҲҗзҶҹпјҢйҖӮеҗҲе·Іжңү Java/Python жөӢиҜ•ж Ҳзҡ„дјҒдёҡпјӣPuppeteer дё“жіЁ JavaScript/TypeScriptпјҢAPI жӣҙзҺ°д»ЈпјҢдёҺ Node и„ҡжң¬е’ҢеүҚз«Ҝд»“еә“йӣҶжҲҗжӣҙиҮӘз„¶гҖӮ
  • зӣёжҜ” CypressпјҢCypress ејәи°ғгҖҢеңЁжөҸи§ҲеҷЁеҶ…и·‘жөӢиҜ•гҖҚзҡ„ејҖеҸ‘иҖ…дҪ“йӘҢдёҺ Time Travel и°ғиҜ•пјҢжӣҙеҒҸеүҚз«ҜеӣўйҳҹдәӨдә’ејҸ E2EпјӣPuppeteer жӣҙйҖҡз”ЁпјҢйҖӮеҗҲзҲ¬иҷ«гҖҒжү№еӨ„зҗҶгҖҒPDF/жҲӘеӣҫжөҒж°ҙзәҝеҸҠеә•еұӮиҮӘеҠЁеҢ–пјҢиҖҢйқһејҖз®ұеҚіз”Ёзҡ„жөӢиҜ• UI жЎҶжһ¶гҖӮ
  • иҫ№з•ҢиҜҙжҳҺпјҡжң¬д»“еә“дёә Puppeteer жәҗз Ғ monorepoпјӣеә”з”Ёдҫ§йҖҡеёё npm i puppeteer еҚіеҸҜпјҢж— йңҖ clone жң¬д»“еә“гҖӮpuppeteer-core дёҚиҜ»еҸ– Puppeteer й…ҚзҪ®ж–Үд»¶дёҺзҺҜеўғеҸҳйҮҸдёӯзҡ„дёӢиҪҪйҖүйЎ№пјҢйңҖиҮӘиЎҢз®ЎзҗҶжөҸи§ҲеҷЁи·Ҝеҫ„гҖӮ

йҖӮеә”дәәзҫӨ

  • йңҖиҰҒеңЁ Node.js/TypeScript дёӯзј–еҶҷзҲ¬иҷ«гҖҒжҲӘеӣҫгҖҒPDF еҜјеҮәжҲ–иЎЁеҚ•иҮӘеҠЁеҢ–и„ҡжң¬зҡ„еҗҺз«ҜжҲ–е…Ёж Ҳе·ҘзЁӢеёҲгҖӮ
  • з»ҙжҠӨеҹәдәҺ Chrome DevTools Protocol зҡ„ E2E жөӢиҜ•гҖҒи§Ҷи§үеӣһеҪ’жҲ– CI еҶ’зғҹжөҒж°ҙзәҝзҡ„еүҚз«Ҝ/жөӢиҜ•е·ҘзЁӢеёҲгҖӮ
  • иҜ„дј°жөҸи§ҲеҷЁиҮӘеҠЁеҢ–еә•еә§пјҲеҗ« MCP/Agent еңәжҷҜпјүзҡ„жҠҖжңҜиҙҹиҙЈдәәпјҢжҲ–д»Һ Puppeteer иҝҒз§»еҲ° Playwright/Selenium зҡ„йҖүеһӢеҜ№жҜ”иҖ…гҖӮ

еҰӮдҪ•дҪҝз”Ё

еүҚзҪ®жқЎд»¶

  • Node.js 18+пјҲд»Ҙ е®ҳж–№ Troubleshooting дёҺеҪ“еүҚ major зүҲжң¬иҰҒжұӮдёәеҮҶпјӣиҝҮж—§ Node еҸҜиғҪеҜјиҮҙжЁЎеқ—и§Јжһҗй”ҷиҜҜпјүгҖӮ
  • зЈҒзӣҳз©әй—ҙпјҡе®үиЈ… puppeteer дјҡдёӢиҪҪ Chrome for TestingпјҲзј“еӯҳй»ҳи®ӨеңЁ ~/.cache/puppeteerпјүгҖӮ
  • Linux ж— еӨҙзҺҜеўғеёёйңҖйўқеӨ–зі»з»ҹдҫқиө–пјҲеӯ—дҪ“гҖҒжІҷз®ұзӣёе…іеә“пјүпјӣDocker еңәжҷҜи§Ғе®ҳж–№ Docker жҢҮеҚ—гҖӮ

е®үиЈ…ж–№ејҸ

ж–№ејҸдёҖпјҡе®Ңж•ҙеҢ…пјҲеҗ«жөҸи§ҲеҷЁдёӢиҪҪпјү

ж–№ејҸдәҢпјҡд»…еә“пјҢдёҚиҮӘеҠЁдёӢиҪҪжөҸи§ҲеҷЁ

иӢҘ npm/pnpm/Yarn/Bun зӯүеҢ…з®ЎзҗҶеҷЁй»ҳи®Өйҳ»жӯўдҫқиө– install и„ҡжң¬пјҢе®үиЈ…еҗҺйңҖжүӢеҠЁдёӢиҪҪжөҸи§ҲеҷЁпјҡ

npm з”ЁжҲ·д№ҹеҸҜеңЁ package.json дёӯдёә puppeteer й…ҚзҪ® allowScripts д»ҘжҒўеӨҚ postinstall дёӢиҪҪпјҲи§Ғ README иҜҙжҳҺпјүгҖӮ

йҰ–ж¬ЎиҝҗиЎҢ

еҲӣе»ә demo.mjsпјҲжҲ– TypeScript зӯүд»·ж–Үд»¶пјүпјҡ

жү§иЎҢпјҡ

жңүз•Ңйқўи°ғиҜ•еҸҜе°Ҷ launch ж”№дёә { headless: false }гҖӮ

йӘҢиҜҒжҳҜеҗҰжҲҗеҠҹ

  • е‘Ҫд»ӨйҖҖеҮәз Ғдёә 0пјҢжҺ§еҲ¶еҸ°жү“еҚ°зӣ®ж ҮйЎөйқўж ҮйўҳпјҲеҰӮ Example DomainпјүгҖӮ
  • ж—  Could not find expected browser locally зӯүжөҸи§ҲеҷЁзјәеӨұжҠҘй”ҷгҖӮ
  • еҸҜйҖүпјҡиҝҗиЎҢ README дёӯзҡ„е®Ңж•ҙзӨәдҫӢпјҲеҗ« locator дёҺй”®зӣҳдәӨдә’пјүпјҢзЎ®и®Ө API дёҺжң¬ең° Chrome зүҲжң¬еҢ№й…ҚгҖӮ

еёёи§Ғеқ‘ / жіЁж„ҸдәӢйЎ№

  • Install и„ҡжң¬иў«жӢҰжҲӘпјҡзҺ°д»ЈеҢ…з®ЎзҗҶеҷЁеёёй»ҳи®ӨзҰҒз”Ё postinstallпјҢеҜјиҮҙиҝҗиЎҢж—¶жүҫдёҚеҲ°жөҸи§ҲеҷЁпјӣеҠЎеҝ…жү§иЎҢ npx puppeteer browsers install жҲ–й…ҚзҪ® allowScriptsгҖӮ
  • зј“еӯҳзӣ®еҪ•дёҚеҸҜеҶҷпјҡCI/жү“еҢ…еңәжҷҜиӢҘ home зӣ®еҪ•дёҚеҸҜз”ЁпјҢи®ҫзҪ® PUPPETEER_CACHE_DIR жҲ–еңЁ .puppeteerrc.js дёӯжҢҮе®ҡ cacheDirectoryпјҢе№¶йҮҚж–°е®үиЈ…жҲ–жүӢеҠЁ install browsersгҖӮ
  • Linux жІҷз®ұдёҺ rootпјҡеңЁ Docker жҲ– root з”ЁжҲ·дёӢеҸҜиғҪйңҖиҰҒ launch({ args: ['--no-sandbox', '--disable-setuid-sandbox'] })пјӣз”ҹдә§зҺҜеўғеә”дјҳе…Ҳдҝ®еӨҚдҫқиө–дёҺжқғйҷҗпјҢиҖҢйқһй•ҝжңҹе…ій—ӯжІҷз®ұгҖӮ
  • HTTP з«ҷзӮ№еҜјиҲӘжҠҘй”ҷпјҡChrome for Testing еҸҜиғҪеҗҜз”Ё HTTPS-First зӣёе…ізү№жҖ§пјҢи®ҝй—® HTTP URL ж—¶еҮәзҺ° net::ERR_BLOCKED_BY_CLIENTпјӣеҸҜжҢү Troubleshooting зҰҒз”ЁеҜ№еә” feature жҲ–ж”№з”Ё HTTPSгҖӮ
  • Windows дјҒдёҡзӯ–з•ҘпјҡиӢҘ Chrome зӯ–з•ҘејәеҲ¶жү©еұ•пјҢй»ҳи®Ө --disable-extensions еҸҜиғҪеҜјиҮҙеҗҜеҠЁеӨұиҙҘпјҢеҸҜи®ҫ enableExtensions: trueгҖӮ
  • дёҺ Playwright йҖүеһӢпјҡйңҖиҰҒеҶ…зҪ® test runnerгҖҒtraceгҖҒеӨҡжөҸи§ҲеҷЁзҹ©йҳөж—¶пјҢеҸҜе№¶иЎҢиҜ„дј° PlaywrightпјӣPuppeteer жӣҙйҖӮеҗҲиҪ»йҮҸи„ҡжң¬дёҺ DevTools ж·ұеәҰйӣҶжҲҗеңәжҷҜгҖӮ
